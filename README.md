# PokéClicker Utilities

A collection of user scripts and potentially other stuff to do with the game
[PokéClicker](pokeclicker) and [PokéClicker Platinum](pokeclicker platinum). Only contains one script for now; might grow, might
not.

Currently contains:

## User Scripts

- `optimal-vitamins.user.js`  
  Automatically distributes the optimal amounts of vitamins to Pokémon.  
  [Greasy Fork](optimal-vitamins-gf)

## Tools

- `fetch-vitamins.php`  
  Converts CSV files downloaded from companion.pokeclicker.com to JSON files to
  be used with JavaScript.
- `fetch-vitamins.py`
  Python version of fetch-vitamins.php
- `auto-fetch-vitamins.py`
  Automatically downloads region vitamin CSV files from companion.pokeclicker.com,
  converts to JSON files, and then updates optima-vitamins.user.js
